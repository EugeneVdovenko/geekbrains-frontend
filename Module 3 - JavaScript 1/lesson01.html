Задание 4

Самостоятельно разобраться с атрибутами тега script (async и defer).

async
  • неблокирующая загрузка в фоне
  • значения: true/false
  • должен быть указан аттрибут src
  • конец загрузки страницы не зависит от скрипта
  • очередность исполнения не зависит от очередности подключения
  • исполняется по мере загрузки
  • для независимых скриптов (счетчики, реклама)
  • по умолчанию async = "false"
  • для динамических скриптов async = "true"
defer
  • неблокирующая загрузка в фоне
  • значения: true/false
  • должен быть указан аттрибут src
  • конец загрузки страницы зависит от скрипта
  • очередность исполнения зависит от очередности подключения
  • исполняется после загрузки DOM, перед браузерным событием DOMContentLoaded
  • для скриптов, которым важен DOM и порядок выполнения
